Є чотири види дужок '{' '[' '(' '<'
Кожній з них відповідають закривающа дужка '}' ']' ')' '>'

Розширити клас String методом valid? для перевірки рядка який складається тільки з дужок на валідність

Написати тест на метод valid?

# "{(})".valid? => false
# "{{[]".valid? => false
# "{[()]}".valid? => true
# "{[()]}".valid? => true
